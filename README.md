#  ShoppyGlobe - Food Ordering E-commerce App

##  Project Overview
ShoppyGlobe is a React-based food ordering e-commerce application built using Vite.  
It allows users to browse food items, search products, view details, add items to the cart, and place orders.

---

## Features

-  Home Page with styled UI  
-  Food Listing Page with Search & Pagination  
-  Search functionality using Redux  
-  Food Detail Page with quantity selection  
-  Cart Management (Add, Remove, Update Quantity)  
-  Checkout Page with order summary  
-  Order Success Page  
-  404 Not Found Page  
-  Lazy Loading using React.lazy & Suspense  
-  Cart Persistence using localStorage  
-  Fully Responsive UI (Tailwind CSS)  

---

## Tech Stack

- React (Vite)  
- Redux Toolkit  
- React Router DOM  
- Tailwind CSS
